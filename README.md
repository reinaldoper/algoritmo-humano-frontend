# Aplicação feito em NextJs para renderizar o backend do Algoritmo Humano.

## Requisitos:
- Node >= 20

---

## Tecnologias utilizadas:
- NextJs
- tailwind css
- React query
- Context API

### Requisitos para rodar esta aplicação:
[Rodar o backend](https://github.com/reinaldoper/algoritmo-humano-backend)


---

### Como rodar a aplicação:
```bash
git clone git@github.com:reinaldoper/algoritmo-humano-frontend.git

cd algoritmo-humano-frontend

npm install

npm run dev

```

- Criar um arquivo .env na raiz:
```bash
NEXT_PUBLIC_BASE_URL=http://localhost:3001/api
```
---

### A aplicação estara rodando em:
```bash
 http://localhost:3000
```
